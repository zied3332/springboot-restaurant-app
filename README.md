# Spring Boot Restaurant App

A simple **learning project** built to practice Spring Boot, JPA, Hibernate, and REST APIs.

This project is part of my journey to improve backend development skills and understand real-world layered architecture using Spring Boot.

---

## Features
- CRUD operations for entities (Client, Menu, Commande, Restaurant…)
- JPA Repositories with automatic query generation
- Entity relationships using Hibernate/JPA
- Lombok integration to reduce boilerplate code
- MySQL relational database
- Layered architecture (Controller → Service → Repository)

---

##  Tech Stack
- **Java 17**
- **Spring Boot 3.x**
- **Hibernate / JPA**
- **MySQL**
- **Maven**

---

## Purpose of This Project
This is a **learning project** created to:
- Practice building REST APIs with Spring Boot  
- Understand JPA and entity relationships  
- Improve code structure and architecture  
- Develop good habits with Git and GitHub
